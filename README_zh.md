## 技术栈
本项目使用以下技术：
- **Vue.js**: 采用 [Arco Design Pro](https://pro.arco.design/)，一个基于 Vue.js 的企业级产品设计系统。

- **Python**: 使用 [FastAPI](https://fastapi.tiangolo.com/) 构建高性能的API服务。

- **MySQL 8**: 高性能的关系型数据库管理系统。
- **Redis**: 高速的内存数据存储，用于缓存。
